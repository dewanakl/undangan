# Security Policy

The development take all security issues seriously. Please do not make public any uncovered flaws.

## Supported Versions

Currently supported project versions with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| < 2.0.0 | :x:                |
